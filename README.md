# AURA

This repository contains the Phase 1 product definition and delivery plan for AURA.

## Current Focus

Phase 1 defines the smallest meaningful working version of the platform:

- visually impaired students can create support requests
- volunteer students can take those requests
- both sides can communicate within the platform
- academic materials can be uploaded and shared safely
- a coordinator can monitor and intervene when needed

## Documents

- `docs/phase-1-product-scope.md`
- `docs/phase-1-delivery-plan.md`

## Phase 1 Outcome

The first release should behave like an academic support coordination platform, not a broad social network or all-purpose assistance app.
